# Galaxy Images

Place galaxy images in this folder. Images should be named descriptively and have corresponding metadata entries in `/src/data/metadata.json`.

Example filenames:
- `M31-Andromeda.jpg`
- `M33-Triangulum-Galaxy.jpg`
- `NGC5194-Whirlpool-Galaxy.jpg`

Images placed here will automatically appear in the Galaxies gallery at `/astrophotography/deep-sky/galaxies`.
